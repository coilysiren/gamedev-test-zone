# game dev test zone

[![Greenkeeper badge](https://badges.greenkeeper.io/lynncyrin/gamedev-test-zone.svg)](https://greenkeeper.io/)

testing a game dev

## running

### setup

- nodejs + npm already installed

```bash
npm install
```

### dev mode

```bash
npm run start
```

### mock production mode

- heroku cli already installed

```bash
heroku local -p 8081
```
